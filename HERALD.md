# HERALD — weekly app intelligence report

The Herald appends a dated section here every Sunday morning (7:00am GT). Broader scope than the Watchman: company news, EOL watch, M1 migration readiness, watchlist maturity, workflow changes.

Format:
```
## Week of YYYY-MM-DD

### Company / ownership news
- App: event (acquisition, shutdown, leadership, pricing model)

### EOL watch
- App: no commits in N months / last release date / verdict

### Sequoia (Intel) compatibility
- App: current posture on Intel Sequoia, expected drop date

### M1 migration readiness
- App: ARM-native / Rosetta / not-yet / broken

### Watchlist maturity (from grip_future.md)
- App: update on trigger condition

### Workflow-change surface
- App: UI overhaul, shortcut changes, default shifts
```
